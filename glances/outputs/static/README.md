# How to contribute?

In order to build the assets of the Web UI, you'll need [npm](https://docs.npmjs.com/getting-started/what-is-npm).

## Install dependencies

```bash
$ npm install
```

## Build assets

```bash
$ npm run build
```

## Watch assets

```bash
$ npm run watch
```

# Anatomy

```bash
static
|
|--- css
|
|--- html
|
|--- images
|
|--- js
|
|--- public # path where builds are put
```
